# v6.25.0

- [#3680](https://github.com/xmrig/xmrig/pull/3680) Добавлен `armv8l` в список 32-битных воспроизводимых сборок ARM
- [#3708](https://github.com/xmrig/xmrig/pull/3708) Незначительные изменения в JIT-компиляторе для Aarch64 (улучшен выбор инструкций, исключена генерация инструкций, которые прибавляют 0, и т.д.)
- [#3718](https://github.com/xmrig/xmrig/pull/3718) Соло-майнинг: добавлена поддержка хардфорка FCMP++
- [#3722](https://github.com/xmrig/xmrig/pull/3722) Добавлено автоматическое определение CPU на Zen4 (Hawk Point)
- [#3725](https://github.com/xmrig/xmrig/pull/3725) Добавлена поддержка RISC-V с JIT-компилятором
- [#3731](https://github.com/xmrig/xmrig/pull/3731) Добавлена поддержка ОС Haiku
- [#3733](https://github.com/xmrig/xmrig/pull/3733) Добавлено определение для MSVC/2026
- [#3736](https://github.com/xmrig/xmrig/pull/3736) RISC-V: добавлена векторизованная инициализация набора данных
- [#3740](https://github.com/xmrig/xmrig/pull/3740) RISC-V: добавлен векторизованный программный AES
- [#3743](https://github.com/xmrig/xmrig/pull/3743) Linux: добавлена поддержка прозрачных больших страниц (transparent huge pages)
- Улучшена поддержка LibreSSL
- Улучшена совместимость для автоматического включения больших страниц в Linux-системах без поддержки NUMA.
